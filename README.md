Titanic Survival Prediction Dashboard

Overview

This project analyzes the Titanic dataset using Power BI to identify factors that influenced passenger survival.

Tools Used

- Power BI
- DAX
- Data Visualization
- Data Analysis

Dashboard Features

- Total Passengers KPI
- Total Survivors KPI
- Survival Rate KPI
- Gender Filter
- Passenger Class Filter
- Survival Distribution Chart
- Gender-wise Analysis
- Class-wise Analysis
- Age Group Analysis
- Interactive Navigation

Key Insights

- Female passengers had a higher survival rate.
- First-class passengers had better chances of survival.
- Most passengers were between 20–40 years old.
- Passenger class and age significantly influenced survival.
